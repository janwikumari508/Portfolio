# 🌐 Personal Portfolio Website

Welcome to the repository for my personal portfolio! This is a fully responsive website built from scratch to showcase my skills, projects, and academic background as a Computer Science Engineering student.

## 🚀 Live Demo
You can view the live website here: [janwikumari508.github.io/portfolio/](https://janwikumari508.github.io/portfolio/)

## ✨ Features
* **Fully Responsive Design:** Optimized for seamless viewing on desktops, tablets, and mobile devices.
* **Clean UI/UX:** A visually appealing layout focused on readability and user experience.
* **Semantic HTML:** Structured with modern web standards for better accessibility.

## 🛠️ Technologies Used
* **HTML5** - For structuring the website's content.
* **CSS3** - For custom styling, layouts, and responsive design.

## 📂 Folder Structure
* `index.html` - The main homepage structure.
* `style.css` - Custom stylesheets for the visual design.
